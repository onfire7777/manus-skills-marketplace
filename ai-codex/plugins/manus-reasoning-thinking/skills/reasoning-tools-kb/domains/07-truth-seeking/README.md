# Truth-Seeking & Verification

Reasoning tools for distinguishing truth from falsehood and validating claims.

## Domains in This Category

1. [Formal Verification](formal-verification.md) - Proofs, invariants, model checking, formal specifications
2. [Experimental Design](experimental-design.md) - Control groups, randomization, blinding, statistical power
3. [Investigative Journalism](investigative-journalism.md) - Source verification, triangulation, paper trails
4. [Logic & Critical Thinking](logic-critical-thinking.md) - Valid inference, fallacy detection, argument structure

## Common Themes

- Systematically testing claims
- Distinguishing correlation from causation
- Identifying sources of bias and error
- Building chains of evidence
- Recognizing limits of knowledge

## Cross-Domain Tools

Tools that appear across multiple domains in this category:
- Controlled experiments
- Triangulation from multiple sources
- Falsifiability and null hypotheses
- Chain of custody and provenance
- Argument mapping and validity checking
