# Continuous Compliance Standards Reference

## Primary Frameworks

### NIST SP 800-137 — Information Security Continuous Monitoring
- **Publisher**: NIST (2011)
- **Application**: Foundational methodology for continuous monitoring adapted to privacy compliance
- **Key Concepts**: Ongoing awareness, risk management, organizational response

### GDPR Accountability Principle (Art. 5(2))
- **Requirement**: Controller shall be able to demonstrate compliance at any time
- **Application**: Continuous monitoring provides real-time demonstrability vs. point-in-time audits

### ISO 27001:2022 Clause 9 — Performance Evaluation
- **9.1**: Monitoring, measurement, analysis and evaluation
- **9.2**: Internal audit
- **9.3**: Management review
- **Application**: ISMS continuous monitoring requirements extend to PIMS via ISO 27701
