# Genre Templates

ジャンル別の構成テンプレート、推奨スタイルプロンプト、歌詞特性。

## Pop / Indie Pop

### Structure
```
[Short Instrumental Intro]
[Verse 1] (4 lines)
[Pre-Chorus] (2 lines)
[Chorus] (4 lines)
[Verse 2] (4 lines)
[Pre-Chorus] (2 lines)
[Chorus] (4 lines)
[Bridge] (2-4 lines)
[Final Chorus] (4 lines)
[Outro]
```

### Style Prompt Examples
```
indie pop, female vocal, breathy, mid-tempo, acoustic guitar, piano, nostalgic, polished
upbeat pop, male vocal, catchy melodies, synth-driven, 120 BPM, bright production
dream pop, ethereal female vocal, reverb-heavy, shimmering guitars, slow-tempo
```

### Lyric Characteristics
- キャッチーなフック、繰り返しやすいコーラス
- 普遍的な感情テーマ（恋愛、自己発見、希望）
- 音節数を揃えてメロディに乗りやすく
- アドリブ `(ooh)`, `(yeah)` で空間を埋める

## Hip-Hop / Rap

### Structure
```
[Short Instrumental Intro]
[Verse 1 | rap] (8-16 lines)
[Hook] (4 lines)
[Verse 2 | rap] (8-16 lines)
[Hook] (4 lines)
[Bridge | spoken word] (2-4 lines)
[Hook] (4 lines)
[Outro]
```

### Style Prompt Examples
```
gritty East Coast hip hop, deep 808s, snare-heavy, male vocal, aggressive, boom bap
melodic trap, Auto-Tune vocal, atmospheric synths, 140 BPM, dark, spacious
lo-fi hip hop, chill, jazzy samples, vinyl hiss, relaxed flow
```

### Lyric Characteristics
- 複雑な韻パターン（multi-syllable rhyme, internal rhyme）
- ワードプレイ、メタファー、パンチライン
- アドリブ多用: `(uh)`, `(yeah)`, `(skrrt)`, `(ayy)`
- `[Rap]` タグでラップセクション指定
- フローの変化で緩急をつける

## Rock / Alternative

### Structure
```
[Short Instrumental Intro]
[Verse 1] (4 lines)
[Chorus] (4 lines)
[Verse 2] (4 lines)
[Chorus] (4 lines)
[Guitar Solo]
[Bridge] (2-4 lines)
[Final Chorus | high energy] (4 lines)
[Outro]
```

### Style Prompt Examples
```
alternative rock, male vocal, gritty, distorted guitar, driving drums, 130 BPM, raw
indie rock, jangly guitars, lo-fi production, introspective, mid-tempo
post-punk revival, cold synths, angular guitar, baritone vocal, dark
```

### Lyric Characteristics
- 生々しい感情表現、内省的なテーマ
- パワーコードに合うシンプルで力強いフレーズ
- ブリッジで感情的な転換点を作る
- `[Guitar Solo]` でインスト部分を確保

## Electronic / EDM

### Structure
```
[Short Instrumental Intro]
[Verse 1 | soft vocal] (2-4 lines)
[Build-Up] (2 lines)
[Drop]
[Verse 2 | filtered vocal] (2-4 lines)
[Build-Up] (2 lines)
[Drop]
[Breakdown | atmospheric] (2-4 lines)
[Final Drop]
[Outro | fade]
```

### Style Prompt Examples
```
melodic dubstep, female vocal chops, dreamy synth pads, heavy bass, 140 BPM
progressive house, uplifting, arpeggiated synth, euphoric, festival-ready
future bass, glitchy, pitched vocals, warm chords, 150 BPM, colorful
```

### Lyric Characteristics
- 短く反復的な歌詞（ドロップが主役）
- ボーカルチョップ、フィルター効果との相性
- `[Build-Up]`, `[Drop]`, `[Breakdown]` タグが核心
- 歌詞量は少なめ、空間を活かす

## Ballad / R&B

### Structure
```
[Short Instrumental Intro | piano]
[Verse 1] (4-6 lines)
[Pre-Chorus] (2 lines)
[Chorus | emotional] (4 lines)
[Verse 2] (4-6 lines)
[Pre-Chorus] (2 lines)
[Chorus | emotional] (4 lines)
[Bridge | vulnerable] (4 lines)
[Final Chorus | belting] (4 lines)
[Outro | fade]
```

### Style Prompt Examples
```
R&B ballad, smooth female vocal, piano, strings, slow-tempo, intimate, polished
soul ballad, powerful male vocal, gospel harmonies, organ, emotional
neo-soul, warm, Rhodes piano, jazzy chords, breathy vocal, mid-tempo
```

### Lyric Characteristics
- 感情の深さ、脆さ、親密さ
- メリスマ（音の引き伸ばし）: `lo-ove`, `fee-eel`
- ハーモニータグ: `[Harmonies]`, `[Layered Vocals]`
- ダイナミクスの幅（ウィスパー → ベルティング）

## Lo-fi / Chill

### Structure
```
[Short Instrumental Intro]
[Verse 1 | soft] (4 lines)
[Chorus | dreamy] (2-4 lines)
[Instrumental Break]
[Verse 2 | soft] (4 lines)
[Chorus | dreamy] (2-4 lines)
[Outro | fade]
```

### Style Prompt Examples
```
lo-fi hip hop, chill, vinyl hiss, tape-saturated, mellow vocal, jazz samples
bedroom pop, lo-fi, reverb-heavy, acoustic guitar, whispered vocal, nostalgic
chill wave, synth pads, warm, hazy, female vocal, slow-tempo, night vibes
```

### Lyric Characteristics
- ミニマルな歌詞、余白を活かす
- 日常の小さな瞬間を描写
- ウィスパー / ソフトボーカル指定
- テクスチャ重視（`vinyl hiss`, `tape-saturated`）

## J-Pop / Japanese

### Structure
```
[Short Instrumental Intro]
[Verse 1] (4 lines)
[Verse 2] (4 lines)
[Pre-Chorus] (2 lines)
[Chorus] (4 lines)
[Verse 3] (4 lines)
[Pre-Chorus] (2 lines)
[Chorus] (4 lines)
[Bridge] (2-4 lines)
[Final Chorus | high energy] (4 lines)
[Outro]
```

### Style Prompt Examples
```
J-pop, female vocal, bright, synth-driven, catchy, energetic, 130 BPM, polished
anime opening, epic, orchestral, rock guitar, powerful vocal, dramatic, fast-tempo
city pop, 80s Japanese, funky bass, warm synth, groovy, nostalgic, female vocal
```

### Lyric Characteristics
- 日本語歌詞ではひらがな/カタカナの音節数を意識
- サビ（コーラス）の印象的なフレーズが最重要
- Bメロ（Pre-Chorus）の転換感
- 英語フレーズのミックス可能
- アニソン系は感情の起伏が大きい
