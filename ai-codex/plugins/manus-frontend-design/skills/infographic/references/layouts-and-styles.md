# Infographic Layouts and Styles

## Layout

| Value | Best For |
| --- | --- |
| `linear-progression` | process, tutorial, and timeline content |
| `binary-comparison` | before/after, A/B, and pros/cons |
| `hierarchical-layers` | pyramids, ranking, and layered structures |
| `bento-grid` | broad overviews with multiple modules |
| `hub-spoke` | one central concept with surrounding points |
| `dashboard` | metrics, KPIs, and statistics boards |
| `venn-diagram` | overlaps and intersections |
| `winding-roadmap` | journeys, phases, and milestones |
| `circular-flow` | recurring cycles |
| `dense-modules` | dense knowledge cards and high-information summaries |

## Style

| Value | Best For | Recommended Mapping |
| --- | --- | --- |
| `craft-handmade` | warm, handmade, approachable visuals | `watercolor` |
| `chalkboard` | teaching, classroom, and tutorial visuals | `chalk` |
| `corporate-memphis` | flat, modern, operational infographic language | `flat-illustration` |
| `technical-schematic` | technical diagrams, engineering visuals, blueprint feel | `infographic` |
| `bold-graphic` | strong contrast and high visual impact | `poster` |
| `storybook-watercolor` | soft, narrative-oriented visuals | `watercolor` |
| `retro-pop-grid` | retro and trend-driven infographic language | `poster` |
| `minimal` | clean and professional composition | `editorial` |

## Aspect

Recommended defaults:

- `3:4` for dense portrait infographics
- `4:3` for mixed block structures
- `16:9` for landscape infographic slides

## Language

Follow the user's language by default.

If the image contains text, specify the target language explicitly in the prompt and avoid long paragraphs inside the image.
