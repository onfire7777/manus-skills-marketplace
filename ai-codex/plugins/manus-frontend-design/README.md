# manus-frontend-design

Frontend and design: UI/UX, web frontend, design systems, accessibility

**113** skills bundled.

## Install (Codex CLI)

```bash
# clone the marketplace repo
git clone https://github.com/onfire7777/manus-skills-codex.git
# copy this plugin's skills into your Codex skills folder
cp -r manus-skills-codex/plugins/manus-frontend-design/skills/* ~/.codex/skills/
```
