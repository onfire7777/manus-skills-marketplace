# Coordination & Cooperation

Reasoning tools for aligning incentives and enabling collective action.

## Domains in This Category

1. [Mechanism Design](mechanism-design.md) - Incentive compatibility, revelation principles, auction design
2. [Organizational Behavior](organizational-behavior.md) - Team dynamics, organizational culture, decision rights
3. [Constitutional Design](constitutional-design.md) - Separation of powers, checks and balances, democratic institutions

## Common Themes

- Aligning individual and collective incentives
- Designing institutions and rules
- Preventing free-riding and defection
- Enabling information aggregation
- Balancing efficiency with legitimacy

## Cross-Domain Tools

Tools that appear across multiple domains in this category:
- Incentive compatibility
- Common knowledge and coordination games
