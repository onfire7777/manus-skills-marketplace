# Lyric Craft

ソングライティングの技法・原則。Suno AI向けに最適化された実践的ガイド。

## Narrative Structures

### 1. Linear Narrative
時系列で物語が進行する。Country/Folkに最適。
```
Verse 1: 状況設定（いつ・どこ・誰）
Verse 2: 展開（何が起きたか）
Bridge: 転換（気づき・変化）
Final Chorus: 結末（どう変わったか）
```

### 2. Emotional Arc
感情の変化を軸にする。Ballad/Pop に最適。
```
Verse 1: 現状の感情（穏やか/不安）
Pre-Chorus: 感情の高まり
Chorus: 感情の核心（爆発/解放）
Verse 2: 深掘り（なぜそう感じるか）
Bridge: 新しい視点・対比
Final Chorus: 感情の昇華
```

### 3. Snapshot / Vignette
瞬間の切り取り。Lo-fi/Indie に最適。
```
Verse 1: シーン描写（五感で捉えた一瞬）
Chorus: その瞬間の意味
Verse 2: 別の角度からの同じ瞬間
Chorus: 深まった理解
```

### 4. Conversational / Dialogue
語りかけ・対話形式。R&B/Hip-Hop に最適。
```
Verse 1: 片方の視点
Verse 2: もう片方の視点（または内面の対話）
Chorus: 共通のメッセージ
Bridge: 和解・対立の頂点
```

## Rhyme Techniques

### Rhyme Schemes
| Scheme | Pattern | Example |
|--------|---------|---------|
| Couplet | AABB | line1-A, line2-A, line3-B, line4-B |
| Alternate | ABAB | line1-A, line2-B, line3-A, line4-B |
| Enclosed | ABBA | line1-A, line2-B, line3-B, line4-A |
| Free | XAXA | line1-X, line2-A, line3-X, line4-A |

### Internal Rhyme
行の途中に韻を配置:
```
I walk the line, the wine-red sky above
The sound of rain on the pane of my window
```

### Near Rhyme (Slant Rhyme)
完全一致ではない音の近似:
```
love / move, rain / name, time / mine
```
- Suno AIでは完全韻より自然に聞こえることが多い

### Multi-Syllable Rhyme
2音節以上の韻（Hip-Hop で特に重要）:
```
elevation / meditation
beautiful night / dutiful fight
```

## Meter and Syllable Control

### Syllable Counting for Suno
- 各行の音節数を揃えるとメロディが安定する
- 目安: Pop 6-10, Rap 10-16, Ballad 8-12
- 日本語: モーラ数で管理（1行7-15モーラが目安）

### Stress Patterns
英語歌詞では強弱パターンが重要:
```
da-DUM da-DUM da-DUM da-DUM  (iambic)
DUM-da DUM-da DUM-da DUM-da  (trochaic)
```

### Line Length Variation
- 均一な長さは単調になる
- 短い行 → 長い行 → 短い行 のリズムで緩急をつける

## Imagery and Detail

### Show, Don't Tell
| Telling (avoid) | Showing (prefer) |
|-----------------|------------------|
| I'm sad | The coffee's getting cold again |
| I miss you | Your jacket's hanging by the door |
| I'm angry | My fist goes through the drywall |
| I love you | I saved your voicemail from July |

### Sensory Anchors
5つの感覚を織り込む:
- **Visual**: 色、光、形、動き
- **Auditory**: 音、沈黙、リズム
- **Tactile**: 温度、質感、触感
- **Olfactory**: 香り、匂い
- **Gustatory**: 味

### Specific > Generic
| Generic (weak) | Specific (strong) |
|----------------|-------------------|
| a flower | a wilted sunflower |
| a car | a rusted '67 Mustang |
| a song | that Fleetwood Mac record |
| somewhere | the bench by the canal |

## Emotional Dynamics

### Tension and Release
- Verse: テンション蓄積
- Pre-Chorus: テンション上昇
- Chorus: リリース（解放）
- Bridge: 再度テンション→最終リリース

### Contrast Techniques
- Verse（ソフト）vs Chorus（パワフル）
- Bridge で予想外の転換（テンポ、視点、トーン）
- ウィスパー → ベルティングのダイナミクス

### Vulnerability as Strength
- 脆さ、不完全さを恐れない表現
- 質問形で終わるセクションは余韻を生む
- `...` で言い淀みを表現

## Japanese Lyrics Specific

### モーラとリズム
- 日本語は等時的モーラ拍
- 1行のモーラ数を揃えるとメロディに乗りやすい
- 長音（ー）、促音（っ）もモーラとしてカウント

### 日本語の韻
- 母音韻: 同じ母音で終わる（空/風: -a/-e は近似）
- 同母音連続: 「あかり」「さかな」（a-a-i, a-a-a）
- 行末の母音を揃えるだけでも効果的

### 日本語 × Suno AI Tips
- ひらがな・カタカナが読みやすい（漢字は読み間違いリスク）
- 英語フレーズを混ぜるとバリエーション増加
- 感嘆詞: `(ねぇ)`, `(そう)`, `(ああ)` でアドリブ効果
- 文末表現で感情変化: `〜よ`（断定）, `〜ね`（共感）, `〜か`（問い）

## Hook Design

### What Makes a Great Hook
1. **Memorable melody carrier**: 最も歌いやすいフレーズ
2. **Emotional truth**: 聴き手が共感できる核心
3. **Rhythmic punch**: リズム的に際立つ
4. **Repetition-friendly**: 何度聴いても飽きない
5. **Stand-alone meaning**: その行だけで意味が通る

### Hook Placement
- コーラスの1行目が最も強い位置
- タイトルとフックを一致させると記憶に残る
- Pre-Chorusの最後の行でフックを予告する

## Iteration Strategy

### First Draft Mindset
1. 完璧を目指さない。まず書き切る
2. プレースホルダー歌詞でメロディ/構造をテスト
3. 核心のフック/コーラスから書き始める
4. Verseは後から詰める

### Refinement Cycle
1. 構造チェック（セクション長、バランス）
2. フック強度チェック（印象的か？）
3. 韻律チェック（リズム、韻）
4. 具体性チェック（クリシェを排除）
5. Sunoフォーマットチェック（メタタグ、文字数）

### A/B Variant Strategy
- コーラスの2バリエーションを提示
- Verseの視点違い（1人称 vs 3人称）
- テンポ感の異なるスタイルプロンプト
- Suno生成は6回以上のイテレーションが普通
