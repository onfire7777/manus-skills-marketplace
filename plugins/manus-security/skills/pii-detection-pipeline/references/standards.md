# PII Detection Pipeline — Standards References

## Tools and Libraries
- Microsoft Presidio: Open-source PII detection and anonymization (github.com/microsoft/presidio)
- spaCy: Industrial-strength NLP with NER capabilities (spacy.io)
- AWS Macie: Cloud-native data discovery and classification
- Google Cloud DLP: Cloud DLP API for sensitive data detection
- Azure Purview: Data governance with sensitivity labeling

## Standards
- NIST SP 800-188: De-Identifying Government Datasets
- NIST SP 800-122: Guide to Protecting PII Confidentiality
- ISO/IEC 27701:2019: Annex A.7.4.5 (De-identification and deletion)
- Article 29 WP Opinion 05/2014 on Anonymisation Techniques

## PII Classification Standards
- NIST SP 800-122: PII categorization (direct vs. linked/linkable)
- ISO/IEC 29100:2011: PII definition and privacy principles
- GDPR Article 4(1): Personal data definition
- CCPA Section 1798.140(v): Personal information definition
