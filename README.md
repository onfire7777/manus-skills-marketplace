# Manus Skills Marketplace

A Claude Code plugin marketplace bundling **770 skills** into **14 themed plugins**.

## Install

Add the marketplace once:

```bash
/plugin marketplace add onfire7777/manus-skills-marketplace
```

Then install whichever plugins you want:

```bash
/plugin install manus-security     # 134 skills — Security skills: pentesting, exploits, hardening, crypto, AppSec/NetSec
/plugin install manus-privacy     # 55 skills — Privacy and compliance: GDPR/CCPA/HIPAA, DPIAs, AI Act, ISO 27701
/plugin install manus-design     # 113 skills — Frontend and design: UI/UX, web frontend, design systems, accessibility
/plugin install manus-backend     # 55 skills — Backend architecture: APIs, system design, distributed systems, databases
/plugin install manus-devops     # 21 skills — DevOps and infrastructure: CI/CD, containers, cloud, IaC, SRE
/plugin install manus-ai     # 32 skills — AI and prompt engineering: LLM apps, agents, RAG, evals
/plugin install manus-data     # 16 skills — Data analysis: analytics, viz, statistics, ML/data pipelines
/plugin install manus-thinking     # 72 skills — Reasoning and thinking: mental models, critical thinking, epistemics
/plugin install manus-creative     # 23 skills — Creativity and innovation: creative writing, art, ideation
/plugin install manus-strategy     # 68 skills — Planning and strategy: PM, strategic planning, roadmaps, OKRs
/plugin install manus-comms     # 25 skills — Communication: writing, presentations, facilitation, leadership
/plugin install manus-quality     # 46 skills — Quality engineering: code review, testing, debugging, refactoring
/plugin install manus-meta     # 109 skills — Meta-skills: skill creation, learning, productivity
/plugin install manus-domain     # 1 skills — Narrow verticals: medical, legal, finance, scientific domains
```
